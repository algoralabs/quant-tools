# Algora Labs API

## Setup
Install [Pipenv](https://pipenv-fork.readthedocs.io/en/latest/basics.html) File.

```bash
pipenv install
```

## Example
Replace `[username]` and `[password]`.

```bash
python3 ./src/example.py --username=[username] --password=[password] --date=2021-01-13 --asset_class=commodity
```
